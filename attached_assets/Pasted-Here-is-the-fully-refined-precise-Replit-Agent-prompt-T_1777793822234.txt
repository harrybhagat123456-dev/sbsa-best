> ### PARSING LOGIC
> The `.txt` file will contain lines like:
> ```
> 📌 Arithmetic — 591 links
> 📌 Advance/Class PNG — 144 links
> 📌 English/Grammar Study Material — 204 links
> ```
> Extract ONLY the topic name between `📌 ` and ` —` (the em dash `—`, not a regular hyphen `-`).
>
> Use this exact logic per line:
> ```python
> if "📌" in line and "—" in line:
>     name = line.split("📌")[1].split("—")[0].strip()
>     if name:
>         topics.append(name[:128])  # Telegram max is 128 chars
> ```
> Skip all other lines silently. Return the list.
